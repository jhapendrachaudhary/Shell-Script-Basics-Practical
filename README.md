## тЪб How to Run Any Script

```bash
# Step 1 тАФ Clone this repo
git clone https://github.com/YOUR_USERNAME/100_Bash_Scripting_Project.git
cd 100_Bash_Scripting_Project

# Step 2 тАФ Make executable (one script)
chmod +x 01-beginner/hello_user.sh

# Step 3 тАФ Run it
./01-beginner/hello_user.sh

# тФАтФАтФА OR make ALL scripts executable at once тФАтФАтФА
find . -name "*.sh" -exec chmod +x {} \;
```

> **Tip:** Most scripts include a `--help` flag. Try `./script.sh --help` first.

## ЁЯУЛ Bash Quick Cheat Sheet

```bash
# Variables
NAME="world"
echo "Hello, $NAME"

# If / else
if [ -f "file.txt" ]; then
  echo "exists"
else
  echo "not found"
fi

# Loops
for i in {1..5}; do echo "Round $i"; done
while read line; do echo "$line"; done < file.txt

# Functions
greet() { echo "Hey, $1!"; }
greet "Bash"

# Input
read -p "Enter name: " NAME

# Arithmetic
result=$(( 5 + 3 ))

# String checks
[[ -z "$VAR" ]]  # empty?
[[ -n "$VAR" ]]  # not empty?

# Redirect
cmd > out.txt     # stdout to file
cmd 2> err.txt    # stderr to file
cmd &> all.txt    # both to file
```
